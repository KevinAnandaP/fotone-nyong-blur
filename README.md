### 1. Prasyarat
Pastikan Python (versi 3.8 atau yang lebih baru) sudah terinstal.

### 2. Install Pustaka / Dependensi
Buka terminal pada direktori proyek ini dan jalankan perintah:

```bash
python -m pip install -r requirements.txt
```

### 3. Jalankan Program
Jalankan skrip utama dengan perintah:

```bash
python main.py
```
